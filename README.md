Simple utility that allows extinct Sound Blaster USB hardware to work with newer Creative Sound Blaster Control Panel versions.

The utility is compatible with the following hardware:

- SB1300 Recon3D (THX & SBX) ✔️
- SB1090 X-Fi Surround 5.1 ❓
- SB1095 X-Fi Surround 5.1 Pro (THX & SBX) ❓
- SB1240 X-Fi HD (THX & SBX) ❓
- SB1100 X-Fi Go! ❓
- SB1290 X-Fi Go! Pro ❓
- SB1560 Omni Surround 5.1 ❓
- SB1540 R3/A6U ❓

This software requires Windows 10 or above.

[The USB software bundle](https://github.com/andkrau/recon3d_usb_config/releases) includes this software, the final compatible Creative Sound Blaster KS audio driver and the last version of the Sound Blaster Control Panel.
